# 🚀 Hemant Bhosale's Portfolio

Welcome to the repository for my personal portfolio website!

This portfolio is built with modern web technologies, featuring a responsive, interactive, and visually appealing design. It showcases my skills, experiences, and the projects I've worked on.

Live Domain: [hemantbhosale.netlify.app](https://hemantbhosale.netlify.app)

## 🛠 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (React 19)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://gsap.com/)
- **Smooth Scrolling**: [Lenis](https://studiofreight.github.io/lenis/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hemant5241/portfolio-2.0.git
   cd portfolio-2.0
   ```

2. **Install dependencies**
   Make sure you have Node.js installed. We recommend using `pnpm` as the lockfile is generated with it:
   ```bash
   pnpm install
   ```

3. **Run the development server**
   ```bash
   pnpm dev
   ```

4. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the site.
